Detailed Description

A simple image cache.

There are two levels of caching: 1) in memory, 2) on the local file system (in app/Library/Private Documents/Image Cache).

Note:
There currently is no method to flush the file cache.
Warning:
This class is not thread-safe. You should use it from the main thread only.
